ℹ️ These checks require you to inspect the hardware directly or run native OS tools. Use this as a physical inspection checklist alongside the automated scan above.
🌡️
Check CPU / GPU Temperature
Install HWMonitor (free) or HWiNFO64. Idle CPU should be 35–55°C. Under full load <85°C is safe. >95°C = critical.
→ Download: cpuid.com/softwares/hwmonitor.html
🌀
Fan Speed & Noise Check
In HWMonitor, check fan RPM. Laptop fans: 0 RPM at idle (normal), 3000–5000 RPM under load. If fan runs at max RPM constantly → clogged vents. Grinding/clicking sound → fan bearing failing, replace immediately.
→ Listen for: clicking, grinding, rattling = bearing failure
💾
SSD / HDD Health (S.M.A.R.T.)
Install CrystalDiskInfo (free). It reads S.M.A.R.T. data directly from your drive. Shows reallocated sectors, uncorrectable errors, power-on hours, and a health rating. "Caution" or "Bad" = backup immediately.
→ Download: crystalmark.info/en/software/crystaldiskinfo
🧠
RAM Error Check
Windows: press Win+R → type mdsched.exe → restart and test. Or use MemTest86 (bootable USB) for a full test. Errors found = one or more RAM sticks are faulty.
→ Windows Memory Diagnostic: mdsched.exe (built-in)
🔋
Battery Health Report (Laptop)
Open Command Prompt as Administrator → type powercfg /batteryreport → open the HTML file it generates. Compare "Design Capacity" vs "Full Charge Capacity". Below 80% = degraded. Below 50% = replace soon.
→ CMD (Admin): powercfg /batteryreport
🖥️
Motherboard & BIOS Check
Check for blown capacitors (bulging tops on cylindrical components near CPU socket). Run CPU-Z to see motherboard model and BIOS version. Check manufacturer's website for BIOS updates — outdated BIOS causes instability.
→ Visual: look for bulging/leaking capacitors near CPU
🌬️
Dust & Thermal Paste
Every 6–12 months: open case (desktop) or remove back panel (laptop) and blow out dust with compressed air. Thermal paste on CPU: replace every 3–4 years. Old paste cracks and loses conductivity → CPU overheats even with working fan.
→ Tools needed: compressed air, Phillips screwdriver, thermal paste (Arctic MX-4 recommended)
⚡
PSU / Power Check (Desktop)
Random reboots or shutdowns under load = PSU failing. Use HWiNFO64 to check voltage rails (12V, 5V, 3.3V should be within ±5%). Alternatively use a PSU tester. PSU lifespan: 5–10 years.
→ HWiNFO64 → Sensors → look for voltage rail readings
ℹ️ What the browser CAN detect: battery level & charge rate, screen refresh rate, JS heap growth (RAM pressure), CPU core count, network type, and thermal throttling via timing delays. What requires native tools: actual CPU/GPU temperature, fan RPM, S.M.A.R.T. drive data, voltage rails, RAM errors. The automated check combines browser data + your age/symptom inputs + inference rules to estimate hardware health.
